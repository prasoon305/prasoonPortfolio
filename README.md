# Prasoon Shrivastava — Portfolio

Executive portfolio site for Prasoon Shrivastava, Senior Manager – Intelligent Automation.

## 🚀 Deploy to GitHub Pages

### Option 1: New Repository
1. Create a new GitHub repository (e.g. `prasoon-portfolio` or `<your-username>.github.io`)
2. Upload `index.html` to the root of the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose `main` branch → `/ (root)` → Save
6. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

### Option 2: Username Site (no subdirectory in URL)
1. Create a repo named exactly `<your-github-username>.github.io`
2. Upload `index.html` to the root
3. GitHub Pages auto-deploys — no settings needed
4. Live at `https://<your-username>.github.io`
